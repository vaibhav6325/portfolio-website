Web designer

Web design refers to design of websites that are displayed on the internet. It usually refers to the user experience aspect of website development rather than software development....A Web designer works on appearance, layout and in some cases, content of a website

What web-designer do?

Web design identifies the goal of a website or web page and promote accessibility for all potential user. This process involve organizing content and images across a series of pages, integrating application and other interactive element.

The professional who perform this process are called Web designer, and their job include the following duties:

  • Selecting easy to read fonts.
  • Choosing attractive colour scheme that also enable easy to read fonts.
  • Implementing a brand identify into the colour fonts and layout.
  • Creating a map of website structure to ensure intuitive navigation.
  • Placing images logos, text, video, application and their other element.
  • Using coding language such as HTML and CSS to create layout and to style pages.
  • Making optimise version of website and pages both for desktop and mobile viewing.
